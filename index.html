Description

The game is a side scrolling platformed style game. The game is inspired from Robo-Cop and was developed for a UCT Game Design course. The background of the game is basically that you are a ‘super cop’ created by team of scientists to clean up the city. The objective of the game is to kill as many enemies as possible and then reach the citizen at the end once defeating the boss. It only has one level of late. The game does have power ups and health packs which increase your health and unlock weapons.
